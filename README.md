# Sales Performance & Profitability Analysis Dashboard

##  Overview
This project analyzes sales data to uncover insights into revenue performance, customer behavior, and product profitability. The goal is to support data-driven decision-making and improve overall business performance.

##  Tools Used
- Power BI for Data Visualization & Dashboard
- Microsoft Excel for Data Cleaning

##  Dataset
The dataset includes:
- Sales transactions
- Customer demographics which include age, gender and country
- Product categories
- Revenue, cost, and profit metrics

##  Process
- Cleaned and prepared raw data in Excel
- Imported data into Power BI
- Created measures for total revenue, cost, and profit
- Built an interactive dashboard to visualize key metrics

##  Key Insights
- The United States and Australia are the top-performing markets
- The Bikes category generates the highest profit
- Female customers contribute more to overall revenue
- Sales trends show fluctuations, indicating seasonal patterns
- The business is overall profitable

##  Recommendations
- Focus on high-performing markets to maximize revenue
- Improve performance in underperforming regions
- Diversify product categories to reduce dependency
- Target high-value customer segments
- Align strategies with seasonal sales trends
